# Лаборатнорная работа 2: VTK
Мой код на C++ лежит в папке  
```02_vtk/solution```

# Симуляция
Я использовал модель меча и рассмотрел сферически симметричную волну, распространяющуюся от ее кончика.
# Код
```node.hpp``` - файл с определением класса точки
```mesh.hpp``` - файл с определением класса сетки и функции деформации на ней
```gen_mesh.hpp``` - файл, позволяющий генерировать ```.msh``` из ```.stl```
```sim.cpp```- файл для запуска симуляции
```CMakeLists.txt```- файл для сборки проекта
